## 収録作品

『流星のロックマン パーフェクトコレクション』には、ニンテンドーDSで発売されたシリーズ全3作品・計7バージョンが収録されています。

### 流星のロックマン（2006年）
オリジナルDSソフト発売日: 2006年12月14日

| バージョン | 特徴 |
|:---|:---|
| ペガサス | 氷属性・ブレイク性能に強み |
| レオ | 炎属性・パワー系カードに強み |
| ドラゴン | 木属性・回復系カードに強み |

### 流星のロックマン2（2007年）
オリジナルDSソフト発売日: 2007年11月22日

| バージョン | 特徴 |
|:---|:---|
| ベルセルク×シノビ | ベルセルクトライブとシノビトライブが使用可能 |
| ベルセルク×ダイナソー | ベルセルクトライブとダイナソートライブが使用可能 |

### 流星のロックマン3（2008年）
オリジナルDSソフト発売日: 2008年11月13日

| バージョン | 特徴 |
|:---|:---|
| ブラックエース | ブラックエースノイズが使用可能・攻撃重視 |
| レッドジョーカー | レッドジョーカーノイズが使用可能・バランス型 |

## 新機能・追加要素

### オンライン機能

#### オンライン対戦
- インターネットを介した対戦プレイに対応
- **カジュアルマッチ**: 気軽に対戦を楽しめるモード
- **ランクマッチ**: ランキングを競う本格対戦モード
- 当時のDS通信対戦がオンラインで楽しめる

#### オンラインカードトレード
- インターネットを介してバトルカードの交換が可能
- 異なるバージョン間でのカード収集がしやすくなる

### ブラザーバンド機能
- 友達登録機能「ブラザーバンド」を搭載
- **最大100人** まで登録可能
- 登録相手に応じてロックマンの能力が向上
- オンラインでのブラザー登録に対応

### ギャラリーモード
- **豊富なイラスト** を鑑賞可能
- シリーズの歴史を振り返るアーカイブ機能

### ミュージックプレイヤー
- 全作品のオリジナルBGMを自由に鑑賞可能
- **新規アレンジBGM** を追加収録
- 早期購入特典でさらにアレンジ楽曲4曲追加

### バトルカード復刻
- **イベント配布カード**を復刻再現
- **玩具特典カード**も復刻再現
- オリジナル版では入手困難だったレアカードが入手可能に

## 快適性向上機能

### アシスト機能（バトル中）
| 機能 | 説明 |
|:---|:---|
| ロックバスター強化 | ロックバスターの攻撃力を増加 |
| 被ダメージ軽減 | 受けるダメージを軽減 |
| オートヒーリング | 自動回復機能 |

### アシスト機能（フィールド上）
| 機能 | 説明 |
|:---|:---|
| 移動速度アップ | フィールド上の移動速度を向上 |
| エンカウント率調整 | 敵との遭遇率を調整可能（増減どちらも対応） |

### 高解像度フィルター
- ゲーム画面の描画を滑らかにする
- **オフにすればオリジナル版に近いビジュアル** でプレイ可能
- レトロ派からHD派まで両対応

### オートセーブ
- プレイ中のデータを自動保存
- DS版のセーブポイント制から進化

### 画面レイアウト調整
- 表示サイズやレイアウトのカスタマイズ
- DS版の2画面構成を1画面に最適化

## DS版からの主な変更点

### 大幅改善
| 項目 | DS版 | パーフェクトコレクション |
|:---|:---|:---|
| 対戦機能 | ローカル通信のみ | オンライン対戦/カジュアル・ランクマッチ |
| カード交換 | ローカル通信のみ | オンラインカードトレード対応 |
| セーブ | セーブポイント制 | オートセーブ対応 |
| 難易度 | 固定 | アシスト機能で調整可能 |
| 画面 | DS 2画面 | 調整可能な1画面レイアウト |
| グラフィック | ドット表示 | 高解像度フィルター対応 |
| レアカード | 入手困難 | イベント/玩具特典カード復刻 |
| ブラザーバンド | すれ違い通信 | オンライン対応（100人登録可） |

### 継承要素
- ゲーム本編の内容・ストーリーは完全再現
- 各バージョンの違い（独自カード等）も維持
- アクションRPGとしての基本システムは変更なし
